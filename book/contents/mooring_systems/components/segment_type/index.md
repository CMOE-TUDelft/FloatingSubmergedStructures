# Segment Types in Mooring system design

Generally the mooring line is divided into following segments:

Foundation: 
Embedment anchor
Suction or driven pile
Clump weight / Gravity anchor
Connection foundation to floater:
Chain; Studless, Studlink
Steel Spiral strand wire rope
Synthetic wire rope
Floater:
Fairleads / Hull connection
Mooring winches
Connector systems:
In-Line Mooring Tensioner
Pull-in Connector
Intermediate line connectors
